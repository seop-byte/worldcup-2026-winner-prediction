# 2026 FIFA World Cup Performance Prediction

This project predicts the expected performance level of national teams in the 2026 FIFA World Cup using player market values, FIFA rankings, and international match results.

## Dataset
- Football Data from Transfermarkt
- International Football Results: Daily Updates
- FIFA World Ranking

## How to Run
1. Open `worldcup_2026_prediction.ipynb` in Google Colab or Jupyter Notebook.
2. Install required packages using:
   pip install -r requirements.txt
3. Run all cells from top to bottom.
4. The notebook will output model evaluation results and 2026 predicted stage scores.
